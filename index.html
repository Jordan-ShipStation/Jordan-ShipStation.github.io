ShipStation Setup

To begin your Account Setup - Go ahead and Login to your ShipStation Dashboard in a new browser window and  
Follow along with the steps below:

1. Add a Ship From Location

1. Go to Account Settings


2. Click Shipping, then Select Ship From Locations 


3. Click the +Add a Ship From Location button 


4. Enter the Ship From and Return Address Information


5. Click Save Changes

>> Name and Company Name fields are both required

>> You can add as many ship from locations as needed.

4. Select a Label Layout

1. Go to Account Settings


2. Click Printing, then Select Printing Setup


3. Click Document Options next to Labels

4. The label settings window will open, Select the Label layout then click Save Changes


>> Select one of the 4' x 6' Label Layouts if using a Thermal Label printer (i.e Zebra, Dymo, Brother etc)

>> Select one of the 8.5" x 11" Label Layouts if using a Traditional Printer

2. Add a Shipping Provider

1. Go to Account Settings


2. Click Shipping, then Select Carriers & Fulfillment


3. Click the Add a Provider Account button


4. Select the Carrier, then click Connect

5. Follow the Instructions in the Setup Wizard to connect the carrier

5. Connect a Printer

1. Go to Account Settings


2. Click Printing, then Select ShipStation Connect


3. Click the Install ShipStation Connect button


4. Once Installed - Open ShipStation Connect

5. Register your computer - Log in with your ShipStation Username and Password


6. Once Connected, Click Go to ShipStation


7. Mark the Printer(s) you will be using for Labels and Shipping Documents as "Shared"


>> ShipStation Connect will also display and allow you to connect Digital Shipping Scales 

3. Connect a Selling Channel

1. Go to Account Settings


2. Click Selling Channels, then Select Store Setup


3. Click the Connect a Selling Channel or Marketplace button

4. Select the Selling Channel


5. Follow the On-Screen Instructions to complete the connection

(Optional) Branding & Notifications 

1. Go to Account Settings


2. Click Selling Channels, then Select Store Setup


3. Click Edit next to the Selling Channel you would like to update


4. Click the Branding Tab

> Enter the Company name, upload a logo, and any additional settings to your preference

5. Click the Notifications Tab

> Select a Shipment Confirmation email and Delivery Confirmation email if you would like ShipStation to send email notifications

> Select the Marketplace Notification Settings to your preference. (by default the selling channel will be updated automatically when the label is created)
